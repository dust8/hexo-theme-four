# hexo-theme-four

> A simple theme for Hexo

![four screen](four.png)

## Feature
- highlight

## Installation
```bash
$ git clone https://github.com/dust8/hexo-theme-four.git themes/four
```

## Usage
modify site **_config.yml**.

    theme: four
    per_page: 0
    highlight:
      enable: false
